# DebugPanel 重构计划

## 1. 当前状态分析

**文件**: `gui/debug_panel.py` (1658行)

**当前结构**:
- 6个标签页：日志、控制台、下载任务、实验室、信息、调试菜单
- 调试菜单包含5个区块：通知测试、日志级别过滤、变量监控、性能分析、错误捕获

**主要问题**:
1. 违反单一职责原则 - 混合了UI创建、日志管理、控制台执行、变量监控、性能分析、错误捕获等多个职责
2. 字体使用不规范 - 表格组件使用了 monospace_font（像素字体）
3. 功能冗余 - 下载任务、主题色设置等功能不需要
4. 代码组织混乱 - 1600+行代码难以维护
5. 界面风格不统一 - 使用传统的 QTabWidget，不符合主程序的 Fluent Design 风格

## 2. 重构目标

### 2.1 删除的功能
- ❌ **下载任务标签页** - 完全删除（包括 `_create_download_tab` 和相关方法）
- ❌ **实验室主题色设置** - 删除主题颜色、强调色选择器（保留实验室标签页，暂时空白）

### 2.2 修改的功能
- ✅ **字体规范** - 表格组件改用 `body_font()`，只有命令行（控制台）使用 `monospace_font()`
- ✅ **性能分析表格** - 改用 Fluent 风格表格样式（参考 `get_fluent_table_style()`）
- ✅ **界面布局** - 从 QTabWidget 改为顶部图标导航栏 + QStackedWidget 内容区

### 2.3 新增的功能
在"调试工具"标签页中增加以下工具区块：

1. **网络诊断工具**
   - HTTP 连接测试
   - DNS 解析测试
   - 延迟测试
   - 显示当前代理配置

2. **缓存管理工具**
   - 查看缓存大小
   - 清除缓存按钮
   - 缓存文件列表
   - 缓存过期时间设置

3. **日志文件管理**
   - 查看日志文件大小
   - 打开日志文件目录
   - 归档当前日志
   - 设置日志级别

4. **系统信息增强**
   - 显示更多系统信息（CPU、内存、磁盘）
   - 显示 Python 环境信息
   - 显示依赖包版本

5. **快捷操作面板**
   - 重启应用
   - 打开配置目录
   - 打开日志目录
   - 检查更新

### 6.3 导航栏实现
```python
class NavigationBar(QWidget):
    def __init__(self, parent=None):
        super().__init__(parent)
        layout = QHBoxLayout(self)
        layout.setSpacing(20)
        layout.setContentsMargins(20, 10, 20, 10)
        
        # 创建导航按钮
        self.buttons = []
        nav_items = [
            ("日志", "文档.svg"),
            ("控制台", "代码.svg"),
            ("信息", "链接.svg"),
            ("调试工具", "芯片.svg"),
            ("实验室", "灯光.svg"),
        ]
        
        for text, icon_file in nav_items:
            btn = self._create_nav_button(text, icon_file)
            layout.addWidget(btn)
            self.buttons.append(btn)
        
        layout.addStretch()
    
    def _create_nav_button(self, text, icon_file):
        btn = MaterialButton()
        icon_path = f"resources/images/dev/{icon_file}"
        btn.setIcon(QIcon(icon_path))
        btn.setIconSize(QSize(24, 24))
        btn.setFixedSize(60, 60)
        btn.setToolTip(text)
        btn.setAccentColor("#6C757D")  # 未选中
        btn.setOnAccentColor("#2078DA")  # 选中
        return btn
```
