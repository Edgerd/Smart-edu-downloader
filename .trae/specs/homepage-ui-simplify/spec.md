# 主页输入区 UI 优化 Spec

## Why

优化主页 URL 输入区的按钮布局和提示文本，合并功能重复的按钮，简化界面，提升用户体验。

## What Changes

- 将"处理URL"按钮文本改为"智能检索并下载"，并与"下载"按钮合并为一个按钮
- 修改输入框上方提示文本为"请输入智慧教育平台的课本链接一行一个，或课本信息..."
- 删除"浏览器"按钮及对应代码
- 删除"清空"按钮及对应代码
- 修改输入框占位文本为"请输入要智能检索的内容"

## Impact

- 受影响模块: 主页输入区 UI
- 影响代码: `gui/pages/home_page.py`、翻译文件（如有）

## MODIFIED Requirements

### 按钮合并与文本修改

**修改前**:

- "处理URL"按钮 + "下载"按钮（两个独立按钮）

**修改后**:

- 仅保留一个按钮，文本为"智能检索并下载"
- 按钮行为: 同时触发处理URL和下载流程

### 输入框上方提示文本

**修改前**:

```
请输入智慧教育平台的课本链接，一行一个...
```

**修改后**:

```
请输入智慧教育平台的课本链接一行一个，或课本信息...
```

### 输入框占位文本

**修改前**:

```
输入课本URL(一行一个):
```

**修改后**:

```
请输入要智能检索的内容：
```

### 删除按钮

**删除**:

- "浏览器"按钮 — 打开浏览器的功能
- "清空"按钮 — 清空输入框内容

需同时删除:

- 按钮实例定义
- 按钮布局添加代码
- 按钮点击事件绑定
- 按钮相关方法（如无其他地方引用）
