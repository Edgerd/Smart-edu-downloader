## 5.6.17 Beta 6

**更新日期**：2026-07-22

### 新增功能

- 新增一键将项目同步到 GitHub 远程仓库的辅助工具，自动完成仓库检查、提交者信息补全、远程同步、本地提交与推送，并提供网络证书校验问题的跳过选项。
- 新增可通过自然语言指令触发的助手能力，方便随时要求上传当前程序到远程仓库。

## 5.6.17 Beta 5

> **AI 总结**：优化错误提示体验：崩溃日志自动输出到程序根目录，错误弹窗背景跟随主题色并新增捉虫入口，调试面板加入一键闪退功能；同时修复错误提示窗口按钮点击闪退、原生弹窗黑底、水波纹绘制异常崩溃以及主页翻页按钮图标不易辨识的问题。

**更新日期**：2026-07-21

### 新增功能

- 崩溃日志在触发错误提示工具时自动复制到程序根目录，方便直接查看与反馈。
- 错误提示窗口新增“给作者捉虫”按钮，点击后自动检测 github.com 网络连通性；网络通畅时打开 GitHub Issues 反馈页面，网络不佳时自动切换到 B站私信反馈入口。
- 调试面板新增“崩溃测试”卡片，提供“一键闪退”按钮，可在后台线程模拟一次未处理异常，用于验证全局崩溃捕获、崩溃日志输出以及错误提示窗口是否能正常触发。

### 优化改进

- 优化导航栏、侧边栏标签、主按钮等核心按钮的选中状态过渡动画，背景色、文字与图标颜色在选中/取消选中时平滑淡入淡出。
- 实现水波纹覆盖区域内文字与图标即时高亮为白色的效果。
- 状态切换等待当前水波纹动画基本结束后再应用颜色变化，并基于实际动画时长动态估算延迟，避免固定等待导致的迟滞感。
- 错误提示窗口背景色改为跟随当前主题配置，与主程序视觉风格保持一致。

### 修复问题

- 修复崩溃提示窗口在独立子进程中以非模态方式显示时直接闪退的问题，改为模态运行以保证窗口稳定出现。
- 修复 MaterialButton 水波纹动画中引用 `QPen` 但未导入导致的 `NameError`，避免错误提示窗口等场景下的按钮被鼠标点击时引发崩溃或窗口消失。
- 修复错误提示窗口中“复制错误信息”“打开日志目录”“给作者捉虫”“查看详情”等按钮点击后窗口直接消失/闪退的问题，现在这些按钮只会执行对应操作，不会关闭窗口。
- 修复调试面板“一键闪退”确认弹窗背景全黑、按钮样式异常以及显示位置偏移的问题，改用与主程序一致的 Fluent 风格确认对话框并以主窗口为父级居中显示。
- 从全局样式层面统一解决原生 `QMessageBox` 弹窗背景全黑的问题：主程序与崩溃提示子进程启动时均会为 `QMessageBox` 应用白色背景与主题色按钮样式，后续新增 QMessageBox 无需再单独设置颜色。
- 修复主页智能检索翻页按钮图标颜色与按钮背景过于接近导致不易辨识的问题，图标现在固定显示为白色，不再跟随主题色变化。

- 修复主题切换后主页搜索导航图标、Fluent 风格下拉菜单、课本封面占位符、历史记录标签页等未热重载的问题。
- 修复 Windows 系统主题色读取，改用注册表调色板数据按小端序解析当前强调色。
- 彻底修复主页智慧搜索框搜索结果不显示课本封面缩略图的问题：资源列表扁平化时同步提取封面地址，详情接口不可用时自动回退到课本列表接口；通过后台线程获取封面，经信号回到主线程加载，并增加内存缓存与待加载内容编号校验。
- 修复部分教育平台下载时因服务器证书密钥过弱导致 SSL 验证失败的问题：降低 OpenSSL 安全级别并支持失败后跳过验证重试，同时新增 SSL 验证开关。
- 修复调试面板实验室标签页不能滚动、信息标签页内容显示不全的问题。
- 修复程序启动后界面无响应、日志死锁等问题。
- 修复精简模式导航栏按钮文字不显示、圆角半径不一致的问题。
- 修复设置界面控件残留固定色边框、开关控件主题切换后不热重载等问题。
- 修复下载管理页面初始化崩溃、统计区域使用表情符号等问题。
